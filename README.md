# 学生管理系统 Student Management System

一个基于Web的学生管理系统，采用前后端分离架构，支持多角色权限管理。

## 项目特点

- 🔐 基于角色的权限控制（管理员、教师、学生）
- 🌐 前后端分离架构
- 🔒 JWT认证与安全防护
- 📱 响应式用户界面
- 🗄️ MySQL数据库存储

## 技术栈

- **前端**: HTML5, CSS3, JavaScript, Bootstrap
- **后端**: Python Flask
- **数据库**: MySQL
- **认证**: JWT (JSON Web Token)

## 权限说明

- **管理员**: 完整的CRUD权限，可管理所有用户和学生信息
- **教师**: 可查看所有学生信息，编辑部分字段
- **学生**: 只能查看同班同学基本信息和自己的完整信息
